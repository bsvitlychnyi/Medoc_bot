На данный момент этот проект создается для учебных целей

Бот для Telegram, написан на Python 3.7.3
Telegram Bot API - https://core.telegram.org/bots/api#telegram-passport

Возможности:
1) Создание копий баз данных программы M.E.Doc
2) Удаление созданных копий для освобождение места на диске
3) Проверка состояния службы (для сетевой версии программы) и её запуск/остановка
4) Проверка свободного места на диске, где хранятся резервные копии
5) Переодическа проверка появления новой версии программы на сайте разработчика, в случае появления - оповещение администратора
